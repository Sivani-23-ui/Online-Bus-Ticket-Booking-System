# Online Bus Ticket Booking System

## Project Overview
The Online Bus Ticket Booking System is a web-based application that allows users to search, book, and manage bus tickets easily. It provides a simple and user-friendly interface with real-time seat availability.

## Features

User Module:
- User Registration and Login
- Search buses (source, destination, date)
- View seat availability
- Book tickets
- Cancel bookings
- View booking history

Admin Module:
- Add, update, and delete buses
- Manage routes and schedules
- View all bookings
- Manage users

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript
- React.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Installation

1. Clone the repository:
git clone https://github.com/your-username/your-repo-name.git

2. Go to project folder:
cd your-repo-name

3. Install dependencies:
npm install

4. Start backend:
npm start

5. Start frontend:
cd client
npm start

## Usage
Open browser and go to:
http://localhost:3000

Then register/login and start booking tickets.

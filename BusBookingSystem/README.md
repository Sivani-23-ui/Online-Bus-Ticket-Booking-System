# Online Bus Ticket Booking System

## Overview
A Java-based bus booking system with MySQL database integration.

## Tech Stack
- Java
- MySQL
- JDBC (MySQL Connector)

## Setup
1. Clone the repo: `git clone https://github.com/Sivani-23-ui/Online-Bus-Ticket-Booking-System.git`
2. Import into VS Code or IntelliJ.
3. Configure MySQL database and update `DBConnection.java`.
4. Run the project.

## Features
- User registration & login
- Bus search & booking
- Booking history
- Feedback system
